{{ site.title }}

{{ site.title }} ({{ site.data.github.repo }}) is used to notify the user of products and product variations that are low in stock.

Features

Social sharing links appear under product properties on the frontend using the configurations set in the admin.

Management of these social sharing widgets are managed from the backend.

Social links are displayed on the fronend in the footer. The order you specified in the admin is the oder they will display on the frontend.

You have the option to show only the social icon or the icon with the text. Text is not required in the admin.

Current social links are displayed is the order they will appear on the frontend.

Social URL is required. This is a full URL, not a path. Label for the social link is optional. Choose the icon you want to display for the social icon. Avaialbe social icons are from Font Awesome.

Installation

Add this line to your Spree application's Gemfile:

gem '{{ site.data.github.repo }}', github: '{{ site.data.github.path }}'

Run the bundle command to install it:

bundle install

After installing, run the generator:

bundle exec rails g {{ site.data.github.repo }}:install

Testing

Generate a dummy application

bundle exec rake test_app

Running tests

bundle exec rake spec
